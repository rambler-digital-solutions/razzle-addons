## Contributing

### Dev environment

#### Start

To start development you need install `yarn` and deps:

```sh
yarn install
```

#### Code quality

To run linting the codebase:

```sh
yarn lint
```

### Publish

To bump version of changed packages and generate changelog run:

```sh
yarn release
```
